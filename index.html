Projects

BARMER Design System

React Native, WebComponents, TypeScript

The BARMER Design System is a multi-platform design system with over 100 components that work with React DOM and also React Native. All components are implemented with one single codebase and are adapted for multi-platform support using React Native for Web and offer SSR capibilities. The design system also contains all components as WebComponents that work completely framework-agnostic. I built the whole design system from ground up and implemented every single component with React for 3 platforms, Web, iOS and Android. For the WebComponents I used the great micro-library Atomico. While working on this project I contributed a lot to the React Native community, Emotion, Atomico and also published some own libraries.

The iF Design Award Jury App is an internal app, that is used by jurors to rate and comment on product submissions. It's kind of a barcode scanner, where the jurors can scan the barcode of a product submission and then give a rating and/or nominate products for the Design Award. I used React Native and Expo to create the app and used plain JavaScript with Flow type-checking.

iF WORLD DESIGN GUIDE

PHP, Laravel, GraphQL, React, JavaScript, Flow, Sass

The iF WORLD DESIGN GUIDE was a product design platform by the iF Design Award. The platform featured 2,400 design studios, about 3,800 company profiles and more than 500,000 products. The platform was implemented with PHP, Laravel and GraphQL on the backend and mostly vanilla JavaScript and Flow type-checking on the frontend, plus some micro-frontends that I later implemented with React. The products and everything was saved in a PostgreSQL database and as the CMS for editorial content I used Statamic, which is a Git-powered flatfile CMS.
